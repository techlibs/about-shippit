## About Shippit

We are a development company specializing in AI and Blockchain with the mission to make markets more efficient by developing products and projects, as well as providing specialized developers.

Shippit is an AI & Blockchain studio building the next generation of intelligent applications. We combine AI agents, knowledge graphs, and decentralized technologies to create products that think, learn, and scale.

We use modern, AI-native tools that enable rapid development and production-grade reliability:

**Frontend**

- React, Next.js, TypeScript
- Tailwind CSS

**Backend**

**Blockchain**

- Smart contracts, Web3 integration, DApps
